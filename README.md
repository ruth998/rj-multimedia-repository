This project consists of creating a small professional multi-page website using HTML.
The site includes 3 to 5 pages with a correct and accessible semantic structure.
Project Objectives
Use semantic HTML tags (header, nav, main, footer).
Follow accessibility best practices:
   One h1 per page
  * Logical heading order
  * Alt text for images
  * Meaningful links
  * Labels associated with form inputs
  * Use of HTML landmarks

Pages Included
 * Home (index.html)
 * About
 * rojects
 * skills
 * contact
